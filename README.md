# 🏪 Task 2: Store Trial Performance Analysis

## 🔍 Control Store Selection Methodology

Control stores were selected using **Pearson Correlation** during the pre-trial period.

This ensured that stores with similar sales patterns and customer behavior were chosen as control stores, providing a reliable benchmark for evaluating trial performance.

---

## 📍 Trial Stores Evaluated

- Store 77
- Store 86
- Store 88

---

## 📈 Performance Assessment

The following metrics were analyzed:

- Total Sales
- Customer Count
- Transactions per Customer
- Monthly Sales Trends

Statistical testing was conducted to determine whether differences between trial and control stores were significant.

---

## 💡 Key Findings

🚀 Store 88 recorded substantially higher sales than its control store.

📈 The sales difference was statistically significant.

🏆 Store 88 delivered the strongest positive trial outcome among all trial stores.

⚠️ Performance varied across stores, highlighting differences in trial effectiveness.

🎯 Customer behavior metrics provided valuable insights into store performance.
